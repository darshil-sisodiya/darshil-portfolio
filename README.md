# Darshil Sisodiya - Portfolio Website

A modern, sleek portfolio website showcasing Darshil's skills, projects, and professional journey. Built with a focus on clean design, smooth animations, and an exceptional user experience.

## ✨ Features

### 🎨 Design & UI
- **Modern Black & White Theme** - Clean, professional aesthetic with subtle gradients
- **Custom Cursor with Glow Effect** - Unique interactive cursor with white glow and smooth animations
- **Responsive Design** - Fully responsive across all devices and screen sizes
- **Smooth Animations** - Elegant transitions, hover effects, and scroll animations
- **Glassmorphism Elements** - Subtle backdrop blur effects and transparency

### 🚀 Functionality
- **Interactive Navigation** - Smooth scrolling navigation with mobile hamburger menu
- **Project Showcase** - Detailed project cards with hover effects and external links
- **Skills Display** - Organized skill categories with interactive tags
- **Contact Integration** - Direct links to LinkedIn, GitHub, and email
- **Performance Optimized** - Throttled scroll events and efficient animations

### 🎯 Sections
1. **Hero Section** - Eye-catching introduction with floating skill cards
2. **About Me** - Personal description and organized skill showcase
3. **Projects** - Featured projects with detailed descriptions and tech stacks
4. **Contact** - Professional contact methods with hover effects

## 🛠️ Technologies Used

- **HTML5** - Semantic markup and structure
- **CSS3** - Modern styling with Flexbox, Grid, and animations
- **JavaScript (ES6+)** - Interactive functionality and smooth animations
- **Font Awesome** - Professional icons
- **Google Fonts** - Inter font family for clean typography

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Local development server (optional, for development)

1. **Clone or Download** the project files
2. **Open** `index.html` in your web browser
3. **Enjoy** the portfolio experience!

### Development Setup

1. **Navigate** to the project directory
2. **Open** `index.html` in your preferred code editor
3. **Customize** content, colors, and styling as needed
4. **Test** responsiveness across different devices

## 🎯 Key Features Explained

### Custom Cursor
- **Dual Cursor System**: Main cursor and follower cursor
- **Glow Effect**: Subtle white glow with customizable radius
- **Hover States**: Cursor scales and intensifies on interactive elements
- **Smooth Movement**: 60fps animation with easing

### Floating Cards
- **Random Variations**: Each card has unique rotation and scale
- **Staggered Animation**: Cards animate with different delays
- **Responsive**: Adapts to different screen sizes

### Project Cards
- **Hover Effects**: Smooth lift and scale animations
- **Overlay System**: Project links appear on hover
- **Tech Stack Tags**: Organized technology indicators
- **External Links**: Direct navigation to live projects

## 🔍 Performance Features

- **Throttled Scroll Events** - Optimized for 60fps performance
- **Intersection Observer** - Efficient scroll-based animations
- **CSS Transforms** - Hardware-accelerated animations
- **Minimal DOM Manipulation** - Efficient JavaScript execution
